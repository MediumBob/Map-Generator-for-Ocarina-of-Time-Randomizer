This tool assists in navigation of the randomized map generated with the Ocarina of Time Randomizer v7.1 (by TestRunner et. al.) by visualizing the map and providing tools to search for regions and find paths between them.

  • If you would like to map your own seed, you'll need to upload the seed's spoiler file. click the "Browse" button below to upload.

  • If you would just like to see how the tool works, click the "Try Demo!" button below to use the demo spoiler.

How to use this tool:

  1. You will need a randomized seed to use this tool. Generate a seed by navigating to the Ocarina of Time Randomizer Web Generator (by dragonbane0 & TreZc0_) and selecting "Play Now". Adjust the randomization rules as desired, then select "GENERATE SEED!" at the bottom of the page.

  2. On the following page after the tool has finished generating your seed, ensure that the "ROM Generation" tab is selected and then click "BROWSE" to select your base ROM (this will be a file with the .n64 extension). Scroll down and select "PATCH ROM!".

  3. Below the "PATCH ROM!" button will be two other options: "Save unmodified Patch File", and "Save Spoiler Log File".

    4. BE SURE to click "Save Spoler Log File" to download the spoiler BEFORE closing out of this window!


    Once you have saved your spoiler file and patched your ROM, you may close out of the randomizer window.

  4. click the "Browse..." button at the top of this page to upload the spoiler file (this will be a file with the .json extension).

    5. This tool should then generate your map and open it in the central window of this webpage. You can use the tools on the left side of this page to search for a region and highlight that node in the graph, or use the path finder to highlight paths from one node to another.

Other Notes

Spoilers generated with Ocarina of Time Randomizer versions prior to 7.1 are not guarenteed to work. The developers have changed (and may continue to change) the naming conventions for the spoiler information, and the current parser does not include region names from older version of the OoT Randomizer.
